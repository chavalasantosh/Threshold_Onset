## Why This Works

### The Key Principle

**Structure = Semantics**

When structure emerges from token patterns, semantics emerge with it. No need for embeddings or neural networks.

### How It Works

1. **Tokenization** (santok): Text → Tokens
   - Multiple methods: whitespace, word, character, grammar, subword, byte
   - Pure Python, standard library only

2. **Token-to-Residue** (adapter): Tokens → Numeric Residues
   - Hash-based conversion (SHA256)
   - Phase 0 compliant: numeric, opaque, no meaning
   - Deterministic: same token → same residue

3. **Structure Emergence** (threshold_onset): Residues → Structure
   - Phase 0: Actions → Residues
   - Phase 1: Segmentation (boundaries, clusters)
   - Phase 2: Identity (persistent segments)
   - Phase 3: Relation (connections between identities)
   - Phase 4: Symbol (aliases, reversible)

4. **Semantics Emerge**: Structure → Semantics
   - Structure reveals meaning
   - Relations reveal semantics
   - No embeddings needed
