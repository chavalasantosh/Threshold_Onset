## Cluster Structure

The system organizes into 4 distinct necessity-shapes:

1. **High Pressure + High Freedom + Distributed** (1 identity)
   - Under constraint, has many escape options
   - Uses all escape paths

2. **Low Pressure + Low Freedom + Concentrated** (3 identities)
   - No constraint pressure, single escape path
   - Most common pattern

3. **Low Pressure + Medium Freedom + Distributed** (1 identity)
   - Moderate pressure, 2 escape paths
   - Distributed use

4. **Medium Pressure + Low Freedom + Concentrated** (1 identity)
   - Some pressure, single escape path
   - Concentrated use

## Key Patterns

### Pattern 1: Pressure-Freedom Relationship
- **High pressure** → **High freedom** (Symbol 4)
- **Low pressure** → **Low freedom** (most identities)
- **Medium pressure** → **Low freedom** (Symbol 2)

**Observation:** Pressure and freedom are not inversely related. High pressure can have high freedom.

### Pattern 2: Concentration Distribution
- **3 identities:** Fully concentrated (concentration = 1.0)
- **2 identities:** Distributed (concentration < 0.5)
- **1 identity:** Mixed (concentration between 0.5-1.0)

**Observation:** Most identities are fully concentrated (single escape path).

### Pattern 3: Escape Network
- **Hub identity:** Symbol 4 (receives escapes from 5, 25, 2)
- **Secondary hub:** Symbol 20 (receives escapes from 18)
- **Distributor:** Symbol 4 (escapes to 2, 18, 25, 20)

**Observation:** Asymmetric flow - some identities are sinks, some are distributors.
